# 分支管理

**创建分支**

git branch  branchName

**切换分支**

git checkout branchName

**合并分支**

git merge

**列出分支**

git branch

**删除分支**

git branch -b branchName

**分支合并**

git merge

**查看远端分支**

git branch -r 

**创建分支并提交到远端**

git push origin branchName

**建立本地默认提交的分支**

git branch --set-upstream-to=origin/branchName 

**取消对master的跟踪**

git branch --unset-upstream master
